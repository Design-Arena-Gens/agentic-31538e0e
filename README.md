## Agents vs Builders vs Models

This repository contains a Next.js 14 application that explains how AI models, builder platforms, and autonomous agents fit together inside modern product stacks. The content lives in `web/src/app/page.tsx` with Tailwind CSS styling.

### Local development

```bash
cd web
npm install
npm run dev
```

### Production build

```bash
cd web
npm run build
npm start
```

The project is preconfigured for deployment on Vercel and publishes to `https://agentic-31538e0e.vercel.app`.
